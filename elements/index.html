Doodle Tag elements

Current app controls, exported as simple website specimens. Each item below shows one button and a very brief description.

Peers

Glue connect or disconnect control.

Menu Toggle

Opens and closes the main tool menu.

Undo

Undo and redo history control.

Clear Canvas

Clear the current doodle or reset saved sessions.

Export

Export the drawing or share the last macro.

Reset View

Reset or cycle the canvas framing.

Rainbow

Toggle the rainbow drawing tool.

Color Palette

Open the quick color picker.

Stroke Width

Cycle or adjust brush thickness.

Gum

Toggle the erasing tool and its strength.

Air Blower

Displace, remove, or mix strokes.

Orthogonal

Toggle directional snapping.

Square Grid

Toggle the square guide grid.

Hex Grid

Toggle the hex guide grid.

Macro

Record, stop, or reset a macro.

Welcome Tour

Play the built-in welcome tour.

Tool Shouts

Cycle tool banner and shout modes.

Dark Mode

Switch between light and dark mode.

Apple Pencil Mode

Optional iPad-only Pencil or finger mode.
